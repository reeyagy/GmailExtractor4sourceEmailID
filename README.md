# GmailExtractor4sourceEmailID
How to Download:
1. CLick on the green Code button
2. CLick download zip file
3. Click on gmail_extractor_app.html in your files
   
How Your Team Can Use It:
Enter their Gmail address (e.g., john.doe@company.com)
Select their preferred scan scope (inbox, entire mailbox, or time-based)
Click "Start Extraction" — Agent authenticates with their Gmail account
Get results — View stats on threads scanned and unique senders found
Create Google Sheet — One-click export to Google Drive
Or download as CSV — Save locally for further analysis
